# Support

PicoMesh is an early-stage open-source project maintained on a best-effort basis.

## Questions and usage help

Search existing issues first. If the question is not already covered, open a GitHub issue and include:

- Board and microcontroller
- Framework and version
- Transport type and clock/baud rate
- Minimal wiring description
- Minimal code reproducer
- Complete compiler or serial output

Do not post private keys, credentials, unpublished research data, or confidential board files.

## Bug reports

Use the bug-report template. A report is most actionable when it identifies the first PicoMesh version or commit where the problem occurs and includes a repeatable test.

## Security reports

Do not open a public issue for a suspected vulnerability. Follow [SECURITY.md](SECURITY.md).

## Research-specific AETERNA questions

The public PicoMesh issue tracker is only for reusable PicoMesh functionality. Questions about private AETERNA models, datasets, voting experiments, PCB production files, or unpublished algorithms should not be posted here.
