# Code of Conduct

PicoMesh follows the Contributor Covenant Code of Conduct, version 2.1.

We pledge to make participation in this project a harassment-free experience for everyone. Be respectful, constructive, and focused on the technical work. Unacceptable behavior may be reported privately to the repository maintainer.

The full Contributor Covenant 2.1 text is available at https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
